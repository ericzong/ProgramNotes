| -------- | ------- | ----- | ----- |
| cat      | dir     | mount | rm    |
| cd       | echo    | move  | rmdir |
| chdir    | erase   | popd  | sleep |

# 缩写

| Noun or Verb | Abbreviation |

| Get          | g            |
| Set          | s            |
| Item         | i            |
| Location     | l            |
| Command      | cm           |
| Alias        | al           |

| Cmdlet name    | Alias |
| -------------- | ----- |
| `Get-Item`     | gi    |
| `Set-Item`     | si    |
| `Get-Location` | gl    |
| `Set-Location` | sl    |
| `Get-Command`  | gcm   |
| `Get-Alias`    | gal   |
