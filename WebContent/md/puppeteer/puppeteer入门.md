# 概述

Puppeteer 是一个 Node 库，它提供了一个高级 API 来通过 DevTools 协议控制 Chromium 或 Chrome。

使用 [DevTools 协议](https://chromedevtools.github.io/devtools-protocol/) 与浏览器进行通信。

# 安装

```shell
npm i --save puppeteer
yarn add puppeteer
cnpm i puppeteer
```
