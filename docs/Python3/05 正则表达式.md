# 概述

re模块对正则表达式提供支持。

> 正则表达式在Python中的地位通常比在Java中更高，甚至有人认为应该先对正则表达式进行学习，然后再进行Python语言学习。

由于正则表达式中通常包含大量反斜杠，所以，为了避免烦人的转义，常使用原始字符串。

# 方法

| 方法                                            | 说明                                                         | 参数                                                         |
| ----------------------------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| re.compile(pattern, flags=0)                    | 返回编译的模式                                               | flags候选值：<br>re.DOTALL （句点匹配换行）<br/>re.IGNORECASE / re.I（忽略大小写）<br/>re.VERBOSE*（忽略空白符和注释#） |
| re.match(pattern, string, flags=0)              | 完全匹配                                                     |                                                              |
| re.search(pattern, string, flags=0)             | 查找匹配                                                     |                                                              |
| re.sub(pattern, repl, string, count=0, flags=0) | 替换                                                         | repl：替换字符串，可为一个函数<br/>count：替换最大次数，默认0替换所有匹配。 |
| re.findall(pattern, string, flags=0)            | 查找所有匹配，返回值：<br/>1. 无分组（包括单分组），返回匹配字符串列表；<br/>2. 有分组，返回分组元组的列表。 |                                                              |
| re.finditer(pattern, string, flags=0)           | 查找所有匹配，返回迭代器                                     |                                                              |
| re.split(pattern, string, maxsplit=0, flags=0)  | 分割，返回列表                                               |                                                              |
| match.group(num=0)                              |                                                              |                                                              |
| match.groups()                                  | 返回所有分组元组                                             |                                                              |

> 说明：
>
> re.VERBOSE，针对的是复杂的正则表达式。当正则表达式过于复杂时，为了提高可读性，我们倾向于为其添加注释。这时，如果使用了该设置，那么就可以使用一个多行的长字符串来编写模式字符串，并且其中的空白符和注释都将被忽略。这使得我们可以在模式字符串添加适当的换行、缩进和注释等来提高可读性。
